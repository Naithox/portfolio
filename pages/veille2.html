Microsoft mise sur le “vibe working” pour moderniser la production de documents

Microsoft introduit le concept de « vibe working » dans Microsoft 365 Copilot pour transformer la création de documents en expérience conversationnelle. Grâce à l’« Agent Mode », l’IA peut générer et ajuster des formules dans Microsoft Excel, analyser des données ou produire des rapports complets dans Microsoft Word via un simple prompt (« vibe writing »). Microsoft lance aussi Office Agent, intégré à Copilot Chat et appuyé sur les modèles de Anthropic, capable de créer présentations et documents prêts à l’emploi. Déployée d’abord sur le web pour les abonnés Microsoft 365, cette approche vise à faire de l’IA un véritable copilote de productivité, en instaurant un dialogue continu entre l’utilisateur et ses outils bureautiques.
